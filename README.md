# Antigravity Developer Preset 🚀

An elite, hyper-compressed configuration and custom skill set designed to maximize developer output, minimize token cost, and enforce strict senior engineering standards across agentic workspaces.

## 🎯 What it does

Most coding agents burn through API limits writing generic placeholders, incomplete functions, and undocumented code. 

This preset compresses **17 production-grade rules** into a single 1.4KB payload, cutting context overhead while forcing models to execute:
* **Zero placeholders/mocks**: Solutions are deployable by default.
* **Strict version locking**: Pins dependencies dynamically to prevent build failures.
* **Automated sanity steps**: Enforces test creation, security boundary checks, and absolute path standards.
* **Refining workflows**: Elevates weak prompts internally prior to generating code.

## 📦 What's inside

```
├── AGENTS.md                 # 1.4KB compressed global rules
└── skills/                   # Custom execution playbooks
    ├── supabase-setup        # Local/remote migration & schema pipeline
    ├── vercel-deploy         # Multi-env, rollback, & Nuxt configurations
    └── framer-workflow       # Component export, sync, & custom overrides
```

## ⚙️ Installation

1. Copy the rules in `AGENTS.md` and append them to your global configurations:
   - **Antigravity path**: `%USERPROFILE%\.gemini\config\AGENTS.md`
   - **Cursor/Windsurf**: Save as `.cursorrules` or `.agents/AGENTS.md` in your project root.
2. Drag the custom workflows from `skills/` into your configuration folder to extend the agent's programmatic actions.
